ℹ️
Mis on MS Projecti kalender?

Kalender määrab, millistel päevadel ja kellaaegadel töö toimub. MS Project kasutab kalendreid ülesannete kestuse arvutamiseks. Vaikimisi kehtib Standard kalender (tööpäevad E–R, 08:00–17:00), aga saad luua ka oma kohandatud kalendreid.

Samm 01

Ava Change Working Time

Kõik kalendriga seotud seaded asuvad Project vahekaardil. Kliki Change Working Time nupul — see avab peamise kalendriakna.

  1. Ava MS Project ja mine ülemisele menüüribale
  2. Kliki vahekaardil Project
  3. Leia paremal pool nupp Change Working Time
  4. Avaneb dialoogiaken, kus saad hallata kõiki kalendreid
💡 Vihje: Change Working Time nupp asub Project vahekaardil, Properties grupis — paremal pool "WBS" nuppudest.
Samm 02

Vaata Standard kalendrit

Enne uue kalendri loomist tutvu vaikimisi Standard kalendriga. See näitab, millised on projekti baastööajad.

  1. Change Working Time aknas on For calendar rippmenüü — vali Standard (Project Calendar)
  2. Kalender näitab märtsikuu — rohelised päevad on tööpäevad, hallid on nädalavahetused
  3. Kliki mõnel tööpäeval (nt 25. märts) — paremal näed tööaegu
  4. Standard kalender: 08:00–12:00 ja 13:00–17:00
Tööpäevad Esmaspäev – Reede
Hommik 08:00 – 12:00
Pärastlõuna 13:00 – 17:00
Lõuna 12:00 – 13:00 (paus)
Samm 03

Loo uus baaskalender

Kohandatud projekti jaoks on mõistlik luua eraldi kalender. Nii saad muuta ainult selle projekti tööaegu, jättes Standard kalendri puutumata.

  1. Kliki Change Working Time aknas nupul Create New Calendar…
  2. Avaneb dialoogiaken Create New Base Calendar
  3. Sisesta kalendri nimi — näiteks TestKalender
  4. Vali Make a copy of: Standard — nii pärib uus kalender kõik Standard seaded
  5. Kliki OK
✅ Nüüd ilmub TestKalender rippmenüüsse. Kõik muudatused, mida teed, mõjutavad ainult seda kalendrit.
Samm 04

Lisa eripäevad (Exceptions)

Eripäevad võimaldavad määrata konkreetsetele perioodidele erinevaid tööaegu — näiteks lühendatud tööpäevad, puhkused või lisatunnid.

Kuidas lisada eripäev?

  1. Vali kalender (TestKalender), kliki all vahekaardil Exceptions
  2. Sisesta eripäeva nimi — näit. Testday
  3. Kliki Details… nupul
  4. Vali Working times ja sisesta tunnid:
    12:00–14:00 ja 15:00–18:00
  5. Määra korduvus (Recurrence pattern): Daily, every 1 day
  6. Range of recurrence: algus 25.03.26, lõpp pärast 32 kordust
  7. Kliki OK
📌 Pärast salvestamist näed kalendris, et kuupäevad 25. märtsist on märgitud eripäevadena (paksus kirjas). Tööajad on nüüd 12:00–14:00 ja 15:00–18:00.
Samm 05

Rakenda kalender ülesandele

Pärast kalendri loomist saa seda üksikutele ülesannetele määrata. Nii arvestab MS Project just selle ülesande puhul kohandatud tööaegu.

  1. Tee topeltklõps ülesandel (nt Rakenduse nõuete määratlemine)
  2. Avaneb Task Information aken
  3. Mine vahekaardile Advanced
  4. Leia väli Calendar ja vali rippmenüüst TestKalender
  5. Kliki OK
⚠️ Kui ülesanne on Manually Scheduled, ei saa mõnda välja muuta. Lülitu Auto Scheduled režiimile, et kalender mõjutaks kestust automaatselt.
Samm 06

Kuidas kalender mõjutab ajakava?

Kalender on MS Projecti ajakava arvutamise alus. Iga muutus kalenderis mõjutab otseselt ülesannete lõpukuupäevi.

⏱️

Tööpäeva pikkus

Standard kalender = 8h/päev. TestKalendri lühendatud tööpäev (5h) tähendab, et sama ülesanne kestab kalendripäevades kauem.

📆

Lõpukuupäev nihkub

Kui ülesande kestus on 8 päeva ja tööpäev on 5h (mitte 8h), lükkub lõpukuupäev automaatselt edasi — MS Project arvestab seda.

🔗

Seotud ülesanded

Kui ülesandel on järglased (Predecessors), nihkub ka nende algus vastavalt kalenderile. Kriitilise tee arvutus muutub.

👥

Ressursside koormus

Resursi kalender ja ülesande kalender võivad erineda. MS Project arvestab mõlemat — töö toimub ainult kattuvatel tööaegadel.
